text properties:
color — Change text color.
font-family — Choose the font style.
font-size — Set text size.
font-weight — Make text bold or light.
font-style — Italicize or normal text.
text-align — Align text left, center, right, or justify.
text-decoration — Add underline, line-through, etc.
text-transform — Change case (uppercase, lowercase).
line-height — Adjust space between lines.
letter-spacing — Space out characters.
word-spacing — Space out words.
text-indent — Indent the first line.
text-shadow — Add shadow behind text.
white-space — Control how spaces and line breaks behave.
overflow-wrap (word-wrap) — Prevent long words from overflowing.

color: #333333;
font-family: 'Helvetica Neue', Arial, sans-serif;
font-size: 18px;
font-weight: 700;
font-style: italic;
text-align: center;
text-decoration: underline;
text-transform: uppercase;
line-height: 1.6;
letter-spacing: 2px;
word-spacing: 4px;
text-indent: 30px;
text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
white-space: nowrap;
overflow-wrap: break-word; /* word-wrap is an alias */

shape properties:
border-radius — Round corners for softer shapes.
clip-path — Create custom, non-rectangular shapes.
shape-outside — Wrap text around complex shapes.
box-shadow — Add depth with shadows.

border-radius: 15px;
clip-path: circle(50% at center);
shape-outside: circle(50%);
box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.25);



images
width — Sets the width of the image
height — Sets the height of the image
object-fit — Controls how the content of a replaced element (like <img>) fits its box (cover, contain, fill, none, scale-down)
object-position — Sets the alignment of the replaced element inside its box
image-rendering — Controls the scaling algorithm (auto, crisp-edges, pixelated)
filter — Applies visual effects like blur, grayscale, brightness to images
opacity — Controls transparency
background-image — Sets an image as background
background-size — Controls the size of the background image (cover, contain, auto, specific sizes)
background-position — Position of the background image
background-repeat — Repeats or not the background image
background-clip — Specifies the painting area of the background

width: 300px;
height: 200px;
object-fit: cover;
object-position: center;
image-rendering: crisp-edges;
filter: grayscale(50%);
opacity: 0.8;
background-image: url('background.jpg');
background-size: contain;
background-position: top right;
background-repeat: no-repeat;
background-clip: padding-box;
